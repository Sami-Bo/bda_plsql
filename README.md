# Bases de données avancées – Travaux pratiques

Ce dépôt regroupe plusieurs travaux pratiques réalisés autour de concepts avancés en bases de données avec Oracle Database et PL/SQL.

---

## Contenu

- **Optimisation de requêtes SQL** : analyse des plans d’exécution, utilisation d’index et impact des statistiques sur les performances  
- **Programmation PL/SQL** : utilisation de curseurs pour manipuler et mettre à jour des données  
- **Modélisation objet et relationnelle** : création de schémas, types objets et méthodes  
- **Bases de données multimédia et spatiales** : gestion d’images et de données géographiques avec requêtes de proximité  

---

## Objectif

L’objectif est d’acquérir une vision globale des fonctionnalités avancées des SGBD, allant de l’optimisation des performances à la gestion de données complexes.
